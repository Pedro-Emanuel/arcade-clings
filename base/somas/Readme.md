# L3 - @somas - Soma de vários números

## Ação

Faça uma função que receba um vetor e retorne e mmc dos números.

### Entrada

* Um número N (N < 50) seguindo de N números inteiros.

### Saida

* O mmc dos N numeros.

## Exemplos

``` py
#INPUT
4 2 5 4 6
#OUTPUT
30
#END<

#INPUT
3 5 2 4
#OUTPUT
20
#END
```

```py
#INPUT
4 4 8 10 15
#OUTPUT
120
#END
```
