# L2 - É @primo

![_](cover.jpg)

## Ação

Dado um número inteiro, faça uma função que verifique se ele é primo.  

## Entrada

* Um numero inteiro N.

### Saida

* 1 caso N for primo e 0 caso contrário.

## Exemplos

``` py
#INPUT
5
#OUTPUT
1
#END

#INPUT
21
#OUTPUT
0
#END

#INPUT
2
#OUTPUT
1
#END
```
